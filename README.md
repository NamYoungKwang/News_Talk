## How to use

### 1. Clone this repo & install dependencies

Install Node dependencies:

`npm install`


### 2. Start the Fastify server

Launch your Fastify server with this command:

```sh
npm run dev
```

## For Build Generation

Build server with command:

```sh
npm run build
```
